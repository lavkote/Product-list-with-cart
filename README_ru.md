## Добро пожаловать! 👋

Спасибо, что выбрали это практическое задание по фронтенд-разработке.

Задания Frontend Mentor помогают улучшить навыки разработки, создавая реалистичные проекты.

**Для выполнения этого задания необходимо хорошее понимание HTML, CSS и JavaScript.**

## Задание

Ваша задача — реализовать проект списка товаров с полностью работающей корзиной и максимально точно повторить предложенный дизайн.

Вы можете использовать любые инструменты. Данные о товарах находятся в локальном файле `data.json`, который можно использовать для динамического заполнения интерфейса.

Пользователь должен иметь возможность:

- добавлять и удалять товары из корзины;
- увеличивать и уменьшать количество товаров;
- видеть модальное окно подтверждения заказа после нажатия **Confirm Order**;
- сбрасывать выбор после нажатия **Start New Order**;
- видеть адаптивный интерфейс для разных размеров экрана;
- видеть состояния наведения и фокуса для интерактивных элементов.

## Где найти материалы

Макеты находятся в папке `/design` (мобильная и десктопная версии). Все ресурсы находятся в папке `/assets`. Также имеется файл `style-guide.md` с палитрой цветов и шрифтами.

## Использование AI-помощников

Файлы `AGENTS.md` и `CLAUDE.md` содержат инструкции для AI-помощников. Они помогают направлять вас, а не выполнять работу за вас.
